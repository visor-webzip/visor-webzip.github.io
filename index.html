¿Qué es un ZIP?
Un ZIP es un archivo comprimido que agrupa varios archivos (y carpetas) en uno solo. Para usarlo, normalmente se descomprime al abrirlo.
¿Qué es el HTML y dónde lo obtengo?
El HTML es el código de una página web. Si una IA te crea una web, normalmente te da el código y un botón de “Copiar” para copiarlo. En muchos proyectos el archivo principal es index.html y se abre por defecto (normalmente el de la carpeta principal).
