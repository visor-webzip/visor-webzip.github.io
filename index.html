Que se puede subir?

Lo que si funciona

  • Materiales tipo web "normal": paginas HTML con imagenes, videos, sonidos y enlaces internos.
  • Recursos creados con eXeLearning u otras herramientas que generan un ZIP con archivos web.

Lo que no funciona o puede fallar

  • Webs que necesitan "servidor" para funcionar (formularios que guardan datos, bases de datos, PHP, etc.).
  • Enlaces a contenidos externos que exigen iniciar sesion.
  • ZIPs que no tienen un archivo de inicio (index.html).
