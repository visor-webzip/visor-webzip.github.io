¿Qué hace este visor?

Este visor abre un ZIP que contiene una web sencilla (HTML, CSS, imágenes, vídeos, audio y enlaces internos) para que puedas compartirla con tu alumnado sin instalar nada.

Qué puedes subir

  • ZIPs con páginas web estáticas y sus recursos (archivos HTML, imágenes, audio, vídeo, PDF, etc.).
  • Materiales creados con eXeLearning u otras herramientas que exportan a web.
  • Carpetas con un HTML principal (normalmente index.html).

Qué no funciona o puede fallar

  • Webs que necesitan servidor (formularios que guardan datos, bases de datos, PHP o similar).
  • Enlaces externos que requieren iniciar sesión o permisos especiales.
  • ZIPs que no contienen ningún archivo HTML.

Cómo funciona

  1. Prepara tu recurso como una carpeta con páginas web (HTML) y comprímelo en ZIP. Puedes hacerlo en la pestaña “Crear ZIP”.
  2. Sube el ZIP a un servicio con enlace público (Google Drive, Dropbox, Nextcloud, GitHub…).
  3. Comparte el archivo para que cualquiera con el enlace pueda verlo.
  4. Copia la URL y pégala en el campo “Pega aqui el enlace publico”.
  5. El enlace para compartir es permanente, así que puedes distribuirlo al alumnado.

Importante sobre Google Drive

Google Drive limita la descarga directa de archivos grandes (aprox. 25 MB). Si tu ZIP supera ese tamaño, puede fallar. Dropbox, Nextcloud y otros servicios suelen permitir archivos más grandes sin ese límite.
